【行业】智能仓储行业初步梳理

发表于 2017-03-01 | 更新于 2018-01-22 | 分类于 行业观察 | 阅读次数：

智能仓储和相关的机器人行业是未来5-10年国内机器人发展的必然趋势，整个智能仓储行业产业链较长，初步梳理观察是否能找到好的投资切入点。

概念

智能仓储的简单理解是通过机器人、自动化系统和软件平台构建的新型仓储和送料、筛检解决方案。智能仓储是物流体系升级的发展方向和必然趋势。

仓储智能化的意义主要在于：

  • 降低成本：通过自动化系统优化仓储流程模型，通过机器人提高作业效率，因而降低单位仓储的人力和时间成本。
  • 降低运维难度：通过自动化解决方案替代原始的人工仓储或混合仓储模式，减少由人员的不稳定性带来的管理和运维压力。

市场规模

根据2016年统计局数据，中国GDP为74.4万亿，其中物流仓储费用占比为15%左右，折合市场规模在11.2万亿左右。2006-2015年，整个物流仓储费用的复合增速为12%，主要得益于今年来国内基础建设、电商的蓬勃发展和土地成本的增值，从增长趋势来看，近三年来随着上述三项的增长的逐步成熟，增速逐渐下滑，可以认为未来其增速水平基本和GDP增速一致，约6%。

就整个物流的费用构成来看，其运输费用占比51%，保管费用占35.5%，管理费用占13.5%，其中仓储成本的占比呈现逐年缓慢增长的趋势，至2016年达到约4万亿。中国的整体仓储产业资产主要以国有为主，目前国有占比超过60%，民营占比约25%，近年来民营仓储资本呈现增长趋势。

至2016年，中国经营性物流仓储面积超过10亿平方米，就仓储固定资产投资而言，超过6000亿元。而目前国内仓储信息化和智能化的渗透率仍然较低，RFID渗透力不超过20%，自动化流水线和机器人等技术渗透率不超过10%，而以每10000平方米仓库智能化改造成本500万测算，则潜在的市场空间也有约5000亿。

发展历史

就仓储行业的解决方案而言，至少可以分为四个阶段：

  • 人工仓储：纯人力存取和筛检，人工管理。
  • 机械化仓储：采用叉车、传送带等机械辅助存取和筛检，人工管理。
  • 自动化仓储：AGV存取，半自动筛检，人机结合管理。
  • 智能仓储：AGV存取，智能识别，自动筛检，基本实现系统自动任务统筹和管理。

就中国的仓储行业发展而言，因为考虑能否升级的关键在于成本和新方案的成熟度，因而其发展结构基本上为金字塔构型，初级阶段的解决方案渗透率最高，而新的解决方案自上而下进行渗透。从下表的历史阶段中可以看出，随着国内机械和自动化水平的提高，国内仓储行业基本已经过渡到机械化仓储阶段。2010年开始，由于电商物流等新兴模式的快速增长，得以支撑龙头企业进行高成本投入的自动化仓储乃至智能仓储模式升级和实验，成为中国智能仓储发展的核心推动力量。

技术路径

目前我国的仓储发展正处在自动化升级阶段。完整的智能仓储系统，其主要硬件设备包括：AGV机器人、自动货架、自动存取机器人、自动分拣系统。软件系统主要包括自动识别、自主导航、任务并发管理、人机协调等系统。

上下料机器人

严格而言，上下料机器人不属于智能仓储范畴，而更贴近工业制造自动化领域范畴，属于制造-半成品-仓储-制造-成品流程中的衔接环节，用于满足物料在生产线和仓储货架之间的高效自动化转移。

目前上下料机器人主要以关节式机械臂和直角坐标机器人（一体化流水线）为主，控制端主要和生产线的耦合程度较高。


码垛机器人

码垛机器人是专门用于物料堆放的机器人，根据不同物料，可以通过编程和模式识别技术进行针对性码垛方式设计，提高效率。码垛机器人主要由大型机械臂和导轨模块组成。

AGV机器人

AGV机器人是工业制造和仓储领域中的主要自动化搬运工具，用于物料、货架或码垛在不同场地之间的运输，根据载重不同，可以分为AGV小车、智能牵引车和智能叉车几类。AGV机器人主要由运动模块，承重模块、导航模块和任务统筹模块构成。


##自动化托盘仓储系统
自动化托盘仓储系统是智能仓储实现中的一种解决方案，其主要思路是直接进行货架改造，以实现货物在货架存储和出入货之间的自动化存取功能，这种结构的存储系统可以直接和前后端分拣机器人合作，从而简化了AVG搬运的步骤。目前该系统的解决方案主要由堆垛机、旋转货架、多层穿梭车等。


分拣机器人

分拣机器人用于货物入库前和出库后的分类和打包，之前主要由人工完成，目前基于传感器、模式识别和自动化技术，已经有若干种解决方案，如交叉带分拣机、翻盘分拣机器人等。


WMS系统

WMS是仓库管理系统，向上对接进销存系统、ERP系统等，核心功能是对仓储作业流程的统筹、指导和规范，完成仓储的数据信息化、增加仓储的流程效率、提高管理透明度和有序程度，是整个智能仓储系统的上层控制和调度系统。

核心技术支持

核心技术是指智能仓储系统中起到关键作用的技术，并构成行业的竞争壁垒。

  • 机器人核心模组：机器人核心模组包括电机、控制器、减速器三类，主要影响机器人的控制精度和运行稳定性，这些核心模组也是机器人成本的主要来源，目前存在巨大的国产替代空间
  • 自动化技术：自动化技术包括电气控制、伺服驱动、反馈调节、PLC控制等方面，是自动化系统构建的基石，目前自动化技术比较成熟，主要的替代率的提高的驱动因素更多在于改造成本。
  • 室内导航：导航系统是AGV等搬运机器人于环境交互的主要方式，主要分为场地装置和机器人本体传感器和软件算法三个层次，在设计时也遵循三位一体的方式，室内导航目前主要分为贴标、视觉、雷达等几种解决方案，实际使用则根据成本和精度需求做权衡。
  • 路径规划：路径规划是AGV等搬运机器人运动自动化运行的基础，主要核心在于算法优化，提高机器人运行效率。
  • 统筹系统：统筹系统是智能仓储的上层控制单元，是针对目前仓储行业自动化程度提高、并发量提高的系统优化，优秀的统筹系统能够提高整个系统运行的效率、能效比和运行时间。统筹系统能够由大数据分析、动态优化或机器学习等技术进行深度优化，是目前智能仓储行业的核心竞争力之一。

产业链

根据智能仓储的软硬件需求，整个产业链包含以下厂商：

  • 元器件、功能模组生产商：产业链上游，主要为下游提供室内导航、机器人减速器、伺服电机、自动化PLC等核心元器件和模组。
  • 机器人、自动化设备生产商：产业链上游，主要提供AGV、输送机、分拣机器人、自动化堆垛机、自动化托盘仓储系统、叉车等硬件产品。
  • 软件系统提供商：产业链上游，为整个仓储硬件提供通用化的控制和调度系统，一般同硬件厂商和系统集成商关系紧密。
  • 系统集成商：产业链中游，直接面向需求客户，根据下游仓储需求，进行软硬件整合、系统定制和ERP、MES数据对接，提供解决方案。
  • 智能仓储独立运营商：以自有仓库为仓储需求方提供仓储服务的独立第三方，是近年来物流仓储行业兴起的主流模式。
  • 直营商：自建自营仓储的需求方，包括电商、零售、汽车、医药等厂商。

市场格局

根据上述的智能仓储产业链（不包括需求方），我们可以确认这个一个典型的B2B市场。在这个市场中，有两种典型的经营模式：

  • 产品销售：针对模组、机器人本体、自动化元器件厂商而言，通常都是采用采购-生产-销售的典型制造业模式，按照产品参数性能区分不同的细分市场，以跑量扩大市占率。
  • 项目制解决方案提供：针对中游的系统集成商而言，一般是项目服务形式经营，以目标客户的质量和数量决定收入规模。

结论

  • 中国仓储行业的规模超过万亿，属于发展平稳的大市场，但智能化和自动化程度低，仅仓储产业升级就存在数千亿的市场空间。
  • 随着人力成本和管理成本的上升，自动化、机器人技术的逐步成熟，仓储自动化和智能化是物流体系发展的必然趋势。
  • 中国仓储自动化进程将由民营行业龙头主力推进，逐步替代，增速较为平稳，直至其成本展示出明显优势后进入快速发展阶段。
  • 随之仓储自动化和智能化水平的提高，其解决方案复杂的提升，产业链变长，从而催生解决方案提供商的市场机会。
  • 较高技术含量的机器人核心部件、自动化模块、视觉导航、智能统筹等细分板块都存在广泛的国产替代机会。
